## 프로젝트 소개

**HERizon**은 **경력 단절 여성**이 다시 일할 수 있도록 돕는  
**온라인 멘토링 & 커뮤니티 플랫폼**입니다.

- **대상**: 30~40대 재취업 희망 여성
- **핵심 키워드**: Empowerment, Mentoring, Community
- **서비스 목적**:
  - 재도약을 위한 **1:1 멘토링 연결**
  - 감정과 경험을 나눌 수 있는 **공감 커뮤니티**
  - 활동에 대한 **포인트 리워드 시스템**
